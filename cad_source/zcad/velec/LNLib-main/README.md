# LNLib-main

## Назначение

Биндинг к нативной геометрической библиотеке LNLib для работы с NURBS-кривыми и поверхностями. Обеспечивает кроссплатформенную динамическую загрузку библиотеки LNLib из C++ в FreePascal.

## Что содержит

```
LNLib-main/
└── dotnet/
    └── CAPI/
        └── include/
            ├── *.h         # C-заголовки API функций
            └── pas/        # Pascal-биндинги
```

### Основные компоненты
- C-заголовки для всех функций LNLib (NurbsCurve, NurbsSurface, BezierCurve и др.)
- Pascal-биндинги с динамической загрузкой
- Загрузчик библиотеки с обработкой ошибок

## Взаимодействие с другими модулями

**Использует:**
- `dynlibs` — динамическая загрузка библиотек
- `uzclog` — система логирования ZCAD

**Используется:**
- `GlobalInterpolation` — для работы с NURBS-кривыми
- Модули, требующие геометрических вычислений

## Важно знать

- Библиотека загружается динамически, не требует статической линковки
- Поддерживает Windows (.dll), Linux (.so), macOS (.dylib)
- Корректно обрабатывает ситуации отсутствия библиотеки
- Все операции логируются через `uzclog`
- Основана на оригинальной библиотеке: https://github.com/BIMCoderLiang/LNLib
