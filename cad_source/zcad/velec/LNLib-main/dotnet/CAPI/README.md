# CAPI

## Назначение

C-совместимый API (Application Programming Interface) для вызова функций библиотеки LNLib из других языков программирования.

## Что содержит

- `include/` — заголовочные файлы C API и Pascal-биндинги

## Взаимодействие с другими модулями

**Использует:**
- Нативную библиотеку LNLib (динамически загружаемую)

**Используется:**
- Pascal-модулями ZCAD через динамическую загрузку

## Важно знать

CAPI использует соглашение о вызовах `cdecl` для кроссплатформенной совместимости.
