# include

## Назначение

Каталог заголовочных файлов C API библиотеки LNLib для работы с NURBS-геометрией.

## Что содержит

### C-заголовки (*.h)
- `NurbsCurve_CAPI.h` — функции работы с NURBS-кривыми
- `NurbsSurface_CAPI.h` — функции работы с NURBS-поверхностями
- `BezierCurve_CAPI.h` — функции работы с кривыми Безье
- `BezierSurface_CAPI.h` — функции работы с поверхностями Безье
- `XYZ_CAPI.h`, `XYZW_CAPI.h` — векторные типы данных
- `Matrix4d_CAPI.h` — матричные операции
- `LNEnums_CAPI.h` — перечисления и константы
- `LNLibDefinitions.h` — базовые определения

### Pascal-биндинги (pas/)
Соответствующие Pascal-модули для всех C-заголовков.

## Взаимодействие с другими модулями

**Используется:**
- `LNLibLoader.pas` — для динамической загрузки функций
- Модулями ZCAD для геометрических вычислений

## Важно знать

- Все функции экспортируются с соглашением `cdecl`
- Pascal-биндинги используют динамическую загрузку через `dynlibs`
- Типы данных совместимы между C и Pascal
