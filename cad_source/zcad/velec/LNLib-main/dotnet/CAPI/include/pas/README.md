# pas

## Назначение

Pascal-биндинги для динамической загрузки функций библиотеки LNLib. Обеспечивают вызов нативных C-функций из FreePascal/Lazarus.

## Что содержит

### Основной загрузчик
- `LNLibLoader.pas` — динамическая загрузка библиотеки с логированием

### Биндинги типов данных
- `XYZ_CAPI.pas` — 3D-вектор (x, y, z)
- `XYZW_CAPI.pas` — 4D-вектор с весом (x, y, z, w)
- `UV_CAPI.pas` — параметрические координаты (u, v)
- `Matrix4d_CAPI.pas` — матрица 4×4
- `LNEnums_CAPI.pas` — перечисления (типы кривых, поверхностей)
- `LNLibDefinitions.pas` — базовые определения и константы

### Биндинги функций
- `NurbsCurve_CAPI.pas` — NURBS-кривые
- `NurbsSurface_CAPI.pas` — NURBS-поверхности
- `BezierCurve_CAPI.pas` — кривые Безье
- `BezierSurface_CAPI.pas` — поверхности Безье
- `Polynomials_CAPI.pas` — полиномиальные функции
- `KnotVectorUtils_CAPI.pas` — работа с узловыми векторами
- `ValidationUtils_CAPI.pas` — валидация данных
- `Projection_CAPI.pas` — проекции
- `Intersection_CAPI.pas` — пересечения
- `LNObject_CAPI.pas` — базовый объект

## Взаимодействие с другими модулями

**Использует:**
- `dynlibs` — стандартный модуль динамической загрузки
- `uzclog` — логирование ZCAD

**Используется:**
- `uGlobalInterpolation` — глобальная интерполяция NURBS
- Другими модулями, требующими геометрических вычислений

## Важно знать

- Все функции загружаются динамически при старте
- Флаг `LNLibLoaded` показывает статус загрузки
- При отсутствии библиотеки выводится предупреждение в лог
- Поддерживаются Windows, Linux и macOS
