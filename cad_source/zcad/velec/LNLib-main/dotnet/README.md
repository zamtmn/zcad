# dotnet

## Назначение

Каталог содержит обёртки для интеграции библиотеки LNLib с управляемыми языками через C API.

## Что содержит

- `CAPI/` — C-совместимый API для вызова функций LNLib

## Взаимодействие с другими модулями

Данный каталог является промежуточным уровнем между нативной библиотекой LNLib (C++) и Pascal-кодом ZCAD.

## Важно знать

Структура следует соглашениям оригинального репозитория LNLib для совместимости с .NET и другими платформами.
