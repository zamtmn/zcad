# test

## Назначение

Каталог тестов для модуля аппроксимации геометрии (Этап 2).

## Что содержит

- `uzvshxtopdfapprogeomtestarc.pas` — unit-тест аппроксимации дуги
- `uzvshxtopdfapprogeomtesterror.pas` — тест численной устойчивости
- `uzvshxtopdfapprogeomtestrender.pas` — визуальный тест рендеринга

## Взаимодействие с другими модулями

**Использует:**
- Модули `approgeom` — тестируемая функциональность
- `uzclog` — логирование

## Важно знать

### Тестовые сценарии
1. Аппроксимация четверти окружности (центр 0,0; радиус 100; угол 0°→90°)
2. Проверка граничных случаев (нулевой радиус, дуга 360°, NaN)
3. Визуальное сравнение (PSNR ≥ 30 dB, SSIM ≥ 0.98)
