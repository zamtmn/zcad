# test

## Назначение

Каталог тестов для модуля ToUnicode CMap (Этап 5).

## Что содержит

- `uzvshxtopdfcmaptesthelper.pas` — вспомогательные функции тестирования
- `uzvshxtopdfcmaptestmapping.pas` — тесты полноты и корректности маппинга
- `uzvshxtopdfcmaptestduplicate.pas` — тест отсутствия дублирования кодов
- `uzvshxtopdfcmapteststream.pas` — тест структуры CMap стрима

## Взаимодействие с другими модулями

**Использует:**
- Модули `cmap` — тестируемая функциональность
- `uzclog` — логирование

## Важно знать

### Тестовые проверки
1. Каждый глиф имеет запись в ToUnicode
2. Корректность Unicode: `A` → `U+0041`, `Б` → `U+0411`
3. Отсутствие дублирующихся кодов символов
4. Наличие всех обязательных элементов CMap стрима
