# test

## Назначение

Каталог тестов для модуля финальной интеграции SHX-шрифтов в PDF (Этап 7).

## Что содержит

Тестовые модули для проверки корректности интеграции:
- Тесты привязки шрифтов
- Тесты генерации текста
- Тесты экранирования символов

## Взаимодействие с другими модулями

**Использует:**
- Модули `geninteg` — тестируемая функциональность
- `uzclog` — логирование результатов

## Важно знать

Тесты запускаются через команду CAD или напрямую из командной строки.
