# test

## Назначение

Каталог тестов для подсистемы кеширования и субсетинга (Этап 6).

## Что содержит

- `uzvshxtopdfsubcachetestdup.pas` — тест дедупликации CharProcs
- `uzvshxtopdfsubcachetesthash.pas` — тест учёта трансформаций в хеше
- `uzvshxtopdfsubcachetestsubset.pas` — тест субсетинга глифов

## Взаимодействие с другими модулями

**Использует:**
- Модули `subcaching` — тестируемая функциональность
- `uzclog` — логирование

## Важно знать

### Тестовые сценарии

**Тест 1: Дедупликация (1000 одинаковых символов)**
- Вход: 1000 запросов символа "A"
- Ожидаемый результат: 1 CharProc, 999 cache hits

**Тест 2: Учёт трансформаций**
- Вход: "A" с height=1.0 и "A" с height=2.0
- Ожидаемый результат: 2 разных хеша, 2 разных CharProc

**Тест 3: Разные шрифты**
- Вход: Font1.shx: "A" и Font2.shx: "A"
- Ожидаемый результат: 2 разных CharProc
