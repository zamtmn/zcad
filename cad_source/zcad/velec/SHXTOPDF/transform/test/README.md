# test

## Назначение

Каталог тестов для модуля трансформаций (Этап 3).

## Что содержит

- `uzvshxtopdftransformtestunit.pas` — unit-тесты примитивов
- `uzvshxtopdftransformtestmatrix.pas` — unit-тесты матриц
- `uzvshxtopdftransformtestpdf.pas` — интеграционные тесты

## Взаимодействие с другими модулями

**Использует:**
- Модули `transform` — тестируемая функциональность
- `uzclog` — логирование

## Важно знать

### Тестовые сценарии

**Unit-тест трансформации примитива:**
- Вход: линия (0,0) → (10,0)
- Применить: height=0.5, widthFactor=2.0, oblique=10°, rotation=45°
- Проверить: координаты результата, отклонение ≤ 0.001

**Unit-тест матриц:**
- Корректность перемножения
- Обратная матрица
- Применение к точке
- Зеркалирование X/Y
