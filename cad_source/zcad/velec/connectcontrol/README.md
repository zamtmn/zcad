# connectcontrol

## Назначение

Модуль управления подключениями электрических устройств. Предоставляет пользовательский интерфейс для анализа и редактирования связей между устройствами на чертеже.

## Что содержит

- `uzvconnect_main.pas` — главный модуль, функция команды
- `uzvconnect_form.pas` — форма пользовательского интерфейса
- `uzvconnect_logic.pas` — бизнес-логика анализа подключений
- `uzvconnect_dwginteraction.pas` — взаимодействие с чертежом
- `uzvconnect_struct.pas` — структуры данных

## Взаимодействие с другими модулями

**Использует:**
- `uzcdrawings` — доступ к текущему чертежу
- `uzccommandsmanager` — регистрация команд
- `uzcinterface` — вывод сообщений пользователю
- `uzclog` — логирование

**Используется:**
- Вызывается пользователем через команду CAD

## Важно знать

### Команда CAD
```
ConnectControl
```

### Последовательность работы
1. Пользователь выделяет устройства на чертеже
2. Выполняет команду `ConnectControl`
3. Модуль анализирует подключения выделенных устройств
4. Открывается форма с результатами анализа

### Требования
- На чертеже должны быть выделены объекты
- При отсутствии выделения выводится предупреждение
