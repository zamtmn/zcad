# connectmanager

## Назначение

Менеджер соединений устройств с поддержкой баз данных (Access, SQLite) и экспорта в Excel. Обеспечивает централизованное управление электрическими подключениями проекта.

## Что содержит

### Корневые модули
- `uzvmanagerconnect.pas` — устаревший менеджер (DEPRECATED)
- `uzvelaccessdbcontrol.pas` — управление Access-базой данных
- `uzvelcontroltempdb.pas` — работа с временной БД
- `uzvelcreatetempdb.pas` — создание временной БД

### Подкаталоги
| Каталог | Назначение |
|---------|------------|
| `core/` | Основная логика (иерархия, менеджер, оптимизатор фаз) |
| `database/` | Работа с базами данных (Access, SQLite) |
| `drawing/` | Взаимодействие с чертежом |
| `excel/` | Экспорт данных в Excel |
| `gui/` | Пользовательский интерфейс |

## Взаимодействие с другими модулями

**Использует:**
- `fpspreadsheet` — работа с Excel-файлами
- `uzcdrawings` — менеджер чертежей
- `uzeentdevice` — устройства на чертеже
- `uzclog` — логирование

**Используется:**
- Подсистемой экспорта данных
- Формами управления устройствами

## Важно знать

- Новая реализация: `core/uzvmcmanager.pas`
- Устаревший код помечен как DEPRECATED
- Поддерживаются Access (.mdb/.accdb) и SQLite
- Интегрируется с GUI через навигационные формы
