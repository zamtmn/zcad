# core

## Назначение

Ядро менеджера соединений. Содержит основную бизнес-логику управления иерархией устройств и оптимизации электрических подключений.

## Что содержит

- `uzvmcmanager.pas` — главный менеджер соединений
- `uzvmchierarchy.pas` — построение иерархии устройств
- `uzvmcphaseoptimizer.pas` — оптимизатор распределения фаз
- `uzvmcstruct.pas` — структуры данных (TVElectrDevStruct)

## Взаимодействие с другими модулями

**Использует:**
- `database/` — работа с БД
- `drawing/` — чтение данных с чертежа
- `uzclog` — логирование

**Используется:**
- `gui/` — отображение данных
- `excel/` — экспорт в Excel
- Командами CAD

## Важно знать

- Это новая реализация, заменяющая `uzvmanagerconnect.pas`
- Структура `TVElectrDevStruct` — основной тип данных устройства
- Оптимизатор фаз обеспечивает равномерное распределение нагрузки
