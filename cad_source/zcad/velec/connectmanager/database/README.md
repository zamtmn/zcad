# database

## Назначение

Модули работы с базами данных для хранения информации об электрических устройствах и их подключениях.

## Что содержит

- `uzvmcaccess.pas` — работа с Microsoft Access (.mdb/.accdb)
- `uzvmcsqlite.pas` — работа с SQLite
- `uzvmcdbconsts.pas` — константы и SQL-запросы

## Взаимодействие с другими модулями

**Использует:**
- Стандартные компоненты доступа к БД
- `uzclog` — логирование

**Используется:**
- `core/` — основная логика менеджера
- Модулями экспорта данных

## Важно знать

- Поддерживаются два типа БД: Access и SQLite
- SQLite предпочтителен для кроссплатформенности
- Access используется для совместимости с существующими проектами
- Константы SQL-запросов вынесены в отдельный модуль
