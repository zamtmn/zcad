# drawing

## Назначение

Модуль взаимодействия менеджера соединений с чертежом ZCAD. Обеспечивает чтение и запись данных устройств на чертеже.

## Что содержит

- `uzvmcdrawing.pas` — основной модуль работы с чертежом

## Взаимодействие с другими модулями

**Использует:**
- `uzcdrawings` — менеджер чертежей ZCAD
- `uzeentdevice` — примитив устройства
- `uzcentcable` — примитив кабеля
- `uzclog` — логирование

**Используется:**
- `core/` — для получения данных с чертежа
- `gui/` — для синхронизации с интерфейсом

## Важно знать

- Модуль является мостом между данными чертежа и структурами менеджера
- Поддерживает чтение атрибутов устройств и кабелей
- Обеспечивает синхронизацию изменений с чертежом
