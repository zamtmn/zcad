# gui

## Назначение

Графический интерфейс пользователя для менеджера соединений. Содержит навигационные формы и компоненты отображения данных.

## Что содержит

- `velectrnav.pas` — главная навигационная форма
- `uzvvstdevpopulator.pas` — заполнение дерева устройств
- `synchmain.pas` — синхронизация данных
- `dbnav.pas` — навигация по базе данных
- `lowvoltnav.pas` — навигация по низковольтному оборудованию
- `specificationnav.pas` — навигация по спецификации

## Взаимодействие с другими модулями

**Использует:**
- `core/` — данные устройств
- `database/` — данные из БД
- LCL/Lazarus — компоненты интерфейса
- VirtualStringTree — древовидное отображение

**Используется:**
- Вызывается из главного меню ZCAD
- Открывается по командам пользователя

## Важно знать

- Использует компонент VirtualStringTree для эффективного отображения больших деревьев
- Поддерживает drag-and-drop для изменения иерархии
- Синхронизируется с выделением на чертеже
