# contolelschema

## Назначение

Модуль управления электрическими однолинейными схемами. Предоставляет главную форму с панелями для работы со списками щитов, фидеров, устройств защиты и информационными данными.

## Что содержит

### Корневой модуль
- `contolelschema_main.pas` — главная форма с разделителями панелей

### Подкаталоги (фреймы)
| Каталог | Назначение |
|---------|------------|
| `listshields/` | Список электрических щитов |
| `listfeeders/` | Список фидеров (отходящих линий) |
| `controldevprotect/` | Устройства защиты |
| `infodata/` | Информационные данные схемы |

## Взаимодействие с другими модулями

**Использует:**
- `uzcguimanager` — менеджер GUI ZCAD
- `Varman` — менеджер переменных для сохранения размеров панелей
- LCL/Lazarus — компоненты интерфейса

**Используется:**
- Открывается из главного меню ZCAD
- Интегрируется с системой чертежей

## Важно знать

- Размеры панелей сохраняются между сессиями
- Каждый фрейм — отдельный функциональный блок
- Панели разделены сплиттерами для изменения размеров
