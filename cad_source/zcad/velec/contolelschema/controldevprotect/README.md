# controldevprotect

## Назначение

Фрейм отображения и управления устройствами защиты (автоматические выключатели, УЗО, дифавтоматы).

## Что содержит

- `contolelschema_controldevprotect.pas` — фрейм устройств защиты

## Взаимодействие с другими модулями

**Использует:**
- Данные о выбранном фидере из `listfeeders`
- Компоненты LCL для отображения

**Используется:**
- `contolelschema_main.pas` — встраивается в главную форму

## Важно знать

- Отображает параметры защитных устройств
- Типы устройств: автоматы (CB), УЗО (RCCB), дифавтоматы (RCBO)
- Позволяет редактировать характеристики защиты
