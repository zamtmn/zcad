# infodata

## Назначение

Фрейм отображения информационных данных электрической схемы (расчётные параметры, сводная информация).

## Что содержит

- `contolelschema_infodata.pas` — фрейм информационных данных

## Взаимодействие с другими модулями

**Использует:**
- Данные из других панелей формы
- Компоненты LCL для отображения

**Используется:**
- `contolelschema_main.pas` — встраивается в главную форму

## Важно знать

- Отображает сводную информацию по выбранным элементам
- Показывает расчётные параметры (токи, мощности)
- Обновляется при изменении выбора в других панелях
