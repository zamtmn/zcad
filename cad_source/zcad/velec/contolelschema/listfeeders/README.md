# listfeeders

## Назначение

Фрейм отображения списка фидеров (отходящих линий) выбранного электрического щита.

## Что содержит

- `contolelschema_listfeeders.pas` — фрейм со списком фидеров

## Взаимодействие с другими модулями

**Использует:**
- Данные о выбранном щите из `listshields`
- Компоненты LCL для отображения

**Используется:**
- `contolelschema_main.pas` — встраивается в главную форму

## Важно знать

- Отображает список отходящих линий щита
- Обновляется при выборе щита в панели `listshields`
- Показывает основные параметры фидеров (номер, нагрузка, защита)
