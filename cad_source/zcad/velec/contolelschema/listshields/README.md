# listshields

## Назначение

Фрейм отображения списка электрических щитов в форме управления однолинейными схемами.

## Что содержит

- `contolelschema_listshields.pas` — фрейм со списком щитов

## Взаимодействие с другими модулями

**Использует:**
- Данные из чертежа о щитовом оборудовании
- Компоненты LCL для отображения

**Используется:**
- `contolelschema_main.pas` — встраивается в главную форму

## Важно знать

- Отображает иерархию щитов проекта
- Позволяет выбирать щит для детального просмотра
- Синхронизируется с другими панелями формы
