# dialux

## Назначение

Модуль экспорта данных освещения в формат DIALux EVO (STF — Standard Transfer Format). Позволяет передавать геометрию помещений и расположение светильников в программу светотехнического расчёта.

## Что содержит

- `uzvdialuxmanager.pas` — главный менеджер интеграции с DIALux
- `uzvcommand_exportdialux.pas` — команда CAD для экспорта
- `uzvspacehierarchy.pas` — иерархия пространств (здание → этаж → помещение)

## Взаимодействие с другими модулями

**Использует:**
- `uzcdrawings` — доступ к чертежу
- `uzeentpolyline` — полилинии (контуры помещений)
- `uzeentdevice` — устройства (светильники)
- `uzclog` — логирование
- `varmandef` — переменные примитивов

**Используется:**
- Вызывается через команду CAD

## Важно знать

### Команда экспорта
```
ExportDialux
```

### Формат STF
- Версия формата: 1.0.5
- Поддерживаемые данные: здания, этажи, помещения, светильники
- Единицы измерения: метры (конвертация из мм)

### Переменные помещений
- `Space_Room` — название помещения
- `space_Floor` — название этажа
- `space_Building` — название здания
- `space_FloorHeight` — высота этажа
