# dialuximport

## Назначение

Модуль импорта результатов светотехнического расчёта из DIALux. Распознаёт светильники из DXF-файла и сопоставляет их с блоками ZCAD.

## Что содержит

- `uzvdialuxlumimporter_main.pas` — главный модуль, функция команды
- `uzvdialuxlumimporter_parser.pas` — парсер DXF/геометрии
- `uzvdialuxlumimporter_recognizer.pas` — распознавание светильников
- `uzvdialuxlumimporter_blocks.pas` — работа с блоками ZCAD
- `uzvdialuxlumimporter_uiform.pas` — форма пользовательского интерфейса
- `uzvdialuxlumimporter_structs.pas` — структуры данных
- `uzvdialuxlumimporter_utils.pas` — вспомогательные функции

## Взаимодействие с другими модулями

**Использует:**
- `uzcdrawings` — доступ к чертежу
- `uzclog` — логирование
- Компоненты LCL — интерфейс

**Используется:**
- Вызывается через команду CAD

## Важно знать

### Команда импорта
```
ImportDialuxLuminaires
```

### Процесс импорта
1. Парсинг геометрии из DXF-файла DIALux
2. Распознавание светильников по геометрическим признакам
3. Поиск соответствующих блоков с префиксом VELEC
4. Отображение формы с результатами сопоставления
5. Вставка блоков в позиции светильников

### Вывод результатов
```
[Dialux Importer] Распознано N светильника(-ов):
  LumKey → (X, Y) [N элемента(-ов)]
```
