# space

## Назначение

Модуль управления пространствами (помещениями) на чертеже. Позволяет создавать, редактировать и визуализировать контуры помещений с расширенными атрибутами.

## Что содержит

- `uzvcommand_spaceadd.pas` — команда добавления пространства
- `uzvcommand_spaceshowhide.pas` — команды показа/скрытия пространств
- `uzvcommand_spaceutils.pas` — вспомогательные функции и общие структуры

## Взаимодействие с другими модулями

**Использует:**
- `uzcdrawings` — менеджер чертежей
- `uzcEnitiesVariablesExtender` — расширение переменных примитивов
- `uzcExtdrIncludingVolume` — вычисление объёмов
- `uzccommand_rectangle` — создание прямоугольников
- `varmandef` — менеджер переменных
- `uzclog` — логирование

**Используется:**
- `dialux/` — для экспорта геометрии помещений
- `lightexchange/` — для сбора данных о пространствах

## Важно знать

### Команды CAD
- `spaceadd` — добавить новое пространство с параметрами

### Атрибуты пространства
Команда поддерживает передачу параметров через операнды:
- Название помещения
- Этаж
- Здание
- Высота
- Другие пользовательские атрибуты

### Слой пространств
Пространства размещаются на отдельном слое для удобства управления видимостью.
