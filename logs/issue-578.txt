# **Техническое задание на разработку команды `layerWeightBigSmall`**

**Путь размещения файлов:**
`zcad\velec\space\`

**Имя основного файла команды:**
`uzvcommand_layerWeightBigSmall.pas`

---

## **1. Назначение команды**

Команда `layerWeightBigSmall(имя_слоя)` предназначена для быстрого переключения веса указанного слоя между двумя фиксированными значениями (большой ↔ маленький).
Используется для автоматизации смены веса линий без необходимости открытия менеджера слоёв.

---

## **2. Формат вызова**

```
layerWeightBigSmall(<LayerName>)
```

`LayerName` — строковое имя слоя.

---

## **3. Логика выполнения**

1. Команда получает имя слоя в качестве аргумента.
2. Проверяет существование слоя в текущем чертеже.
3. Если слой **не существует**:

   * Команда **не выполняется**.
   * В UI через `zcUI` выводится сообщение:
     **"Слой <LayerName> не существует"**.
4. Если слой существует и его текущий вес **не равен CONST_LAYER_WEIGHT_BIG**, то:

   * Установить вес = `CONST_LAYER_WEIGHT_BIG`.
5. Если слой существует и его текущий вес **равен CONST_LAYER_WEIGHT_BIG**, то:

   * Установить вес = `CONST_LAYER_WEIGHT_SMALL`.

---

## **4. Константы**

Все значения веса вывести в константы в этом же файле:

```
CONST_LAYER_WEIGHT_BIG   = 2;
CONST_LAYER_WEIGHT_SMALL = 0;
```

Сделать их легко редактируемыми для будущих изменений (в секции `const`).

---

## **5. Требования к реализации**

### 5.1. Работа со слоями

* Использовать стандартный механизм получения слоя
  (`Drawings.GetCurrentDWG^.GetLayerTable^` или аналог, как в других командах velec).
* Изменение веса слоя производить через его параметр `LineWeight`.

### 5.2. Стиль кода

* Команда должна быть оформлена в едином стиле с другими командами `velec\space`.
* Регистрация команды выполняется в секции `initialization`.
